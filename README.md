Summary: AQC-1 (“Autonomous Quadcopter Version 1”) is a self developed open-source quadcopter project built from scratch. This repository contains the most up to date flight software written in C which is intended to be used on Adafruit's STM32F405 Feather Express: https://www.adafruit.com/product/4382. More information on this quadcopter can be found on my Canvas portfolium: https://portfolium.com/CharlesRoman2.
